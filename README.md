# website-profil-siswa
Tugas SMK Website Sederhana

Tujuan Website

Website kecil ini dibuat untuk:
1. Menampilkan halaman login
2. Setelah login berhasil, masuk ke halaman profil siswa dan sekolah
3. Ada form untuk edit profil
4. Ada menu untuk ubah password
5. Tidak menggunakan database
6. Data disimpan sementara di browser menggunakan localStorage

Di dalam file index.html, ada 3 bagian utama:
HTML  → Struktur halaman (Jadi HTML ibarat kerangka rumah)
CSS   → Tampilan halaman (CSS ibarat cat, dekorasi, dan tata letak rumah)
JS    → Fungsi / logika website (JavaScript ibarat otak atau mesin website)
